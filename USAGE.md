# How to Use Web Security Scanner

--------------------------------------------------
SCANNING A URL
--------------------------------------------------

1. Open the frontend in browser.
2. Enter a website URL.
3. Click "Scan".
4. Wait for analysis.

--------------------------------------------------
RISK SCORE EXPLANATION
--------------------------------------------------

Risk Score Range: 0 – 100

0 – 30   → Low Risk (Safe)
31 – 60  → Medium Risk (Be Careful)
61 – 100 → High Risk (Likely Phishing)

--------------------------------------------------
FEATURES
--------------------------------------------------

- URL validation
- Domain analysis
- SSL certificate checking
- Security headers inspection
- Risk score calculation
- Scan history storage
- Responsive UI

--------------------------------------------------
BEST PRACTICES
--------------------------------------------------

- Always use https://
- Avoid scanning internal/private URLs
- Use API docs for advanced testing
