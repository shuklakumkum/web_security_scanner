\# Web Security Scanner 



A Python-based web security scanner built using FastAPI.

This tool helps analyze websites for common security issues.



\## Features

\- URL validation

\- SSL certificate checking

\- Security headers analysis

\- Response and content inspection



\## Tech Stack

\- Python

\- FastAPI

\- Requests
