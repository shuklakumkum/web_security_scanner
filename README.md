#phishing Detection & Web Security Scanner
A Python-based security tool that identifies fake/phishing websites and analyzes security vulnerabilities. Built with FastAPI

#Features(planned)
**Domain Analysis**
-- URL validation and parsing
-- Typosquatting detection (amaz0n.com vs amazon.com)
-- Domain age and registration check

**Security Scanning**
-- SSL certificate validation
-- Security headers analysis (HSTS, CSP, X-Frame-Options)
-- Mixed content detection

**Phishing Detection**
-- Visual similarity with legitimate sites
-- Suspicious content patterns
-- Risk score calculation

## Tech Stack
-- **Language:** Python 3.11+
-- **Framework:** FastAPI
-- **Validation:** Pydantic
