Web Security Scanner

About

This Web Security Scanner analyzes websites for SSL security, domain safety, security headers, and phishing indicators. It calculates a risk score to help users identify potentially unsafe websites.

Developer: Kumkum

How to Use

  1. Enter a full URL (Example: https://example.com)
  2. Click the Scan button or press Enter
  3. Review the risk score and warnings

Risk Levels

  • Low – Website appears safe
  • Medium – Some security concerns detected
  • High – Potential phishing or unsafe website
